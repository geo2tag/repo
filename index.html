Geo2tag distribution repo

For installing geo2tag do next steps:
  • Add next line into /etc/apt/sources.list file

    deb http://download.geo2tag.org/ geo2tag {testing, stable}

  • Run sudo wget http://download.geo2tag.org/geo2tag.pub
  • Run sudo apt-key add geo2tag.pub
  • Run sudo apt-get update
  • Check by search package "geo2tag": apt-cache search geo2tag
  • Use sudo apt-get install for installing required components
